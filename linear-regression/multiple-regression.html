Multiple Regression

多元线性回归

自变量x多于一个
多元回归模型
y = B0 + B1x1 + B2x2 + ... + Bpxp + e
(e -- Epsilon 是误差值/偏向)

多元回归方程
E(y) = B0 + B1x1 + B2x2 + ... + Bpxp
(和的期望值等于各个单项期望值的和)
(e -- Epsilon是随机变量，期望/均值是0方差对于所有自变量来说相等，所有e的值是独立的，它符合正态分布)

估计多元回归方程
E(y) = b0 + b1x1 + b2x2 + ... + bpxp
B0,B1,B2...Bp点的估计是b0,b1,b2...bp. B0,B1,B2...Bp是真实的值，b0,b1,b2...bp是估计的值。
如果找出简单线性回归模型的最佳回归线 (y_hat = x*b1 + b0) (y_hat也就是估计值--回归线)

依然使用 sum of square, 找其最小值 (求极值和积分)
